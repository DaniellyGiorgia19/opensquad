# Step: Pesquisa de Tendências

## Instruções

1. Leia o brand_profile ativo do negócio e os dados da campanha
2. Use web_search para pesquisar tendências atuais relacionadas ao tema da campanha
3. Identifique pelo menos 3 ângulos de conteúdo diferenciados
4. Compile um briefing completo com fontes, dados e sugestões

## Input
- Brand Profile do negócio ativo
- Dados da campanha (objetivo, público-alvo, mensagem central)

## Output
Salvar briefing de pesquisa em `squads/marketing-pipeline/output/pesquisa-briefing.md`
